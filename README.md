# i3-overlay

**i3-overlay** — это минималистичный монитор системы для i3, отображающий:

- Использование CPU, RAM и диска с анимацией.
- Температуру процессора.
- Сетевой трафик (Upload/Download) и статус интернета.
- Список топовых процессов с возможностью убить процесс прямо из окна.

Окно прозрачное, всегда поверх других и может быть свернуто/развернуто.
